Minimal API Project - DIO

Projeto desenvolvido como parte do desafio prático da DIO, implementando uma API Minimal com .NET 8, Entity Framework e MySQL.

Descrição do Projeto

Esta é uma API Minimal que gerencia veículos, pessoas e administradores com autenticação JWT e autorização por perfis. O projeto demonstra conceitos avançados de .NET e boas práticas de desenvolvimento.

Funcionalidades

- CRUD Completo- Veículos, Pessoas e Administradores
- Autenticação JWT- Sistema seguro de login
- Autorização por Perfis- Adm, Editor e Usuário
- Entity Framework - ORM com MySQL
- Swagger- Documentação interativa da API
- Migrations- Controle de versão do banco
- Validações- DTOs e modelos validados

Tecnologias Utilizadas

- .NET 8*- Framework principal
- Entity Framework Core- ORM para MySQL
- MySQL - Banco de dados relacional
- JWT Bearer - Autenticação e autorização
- Swagger/OpenAPI - Documentação da API
- Git - Controle de versão
